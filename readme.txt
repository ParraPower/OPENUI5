1. Create solution in VS
2. Go to: https://sap.github.io/ui5-tooling/
3. Create packages.json - using npm init
4. ui5 init	
5. Create webapp folder
6. Go to https://help.sap.com/doc/saphelp_uiaddon20/2.05/en-US/3a/9babace121497abea8f0ea66e156d9/content.htm?no_cache=true
7. create manifest.json file in webapp folder
8. copy manifest.json contents from link to manifest.json folder
9. run ui5 use SAPUI5@1.76.0
10. run ui5 add sap.ui.core sap.m themelib_sap_fiori_3
11. ui5 serve
